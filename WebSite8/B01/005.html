30/06/2021: MVC Form Practice (AJAX)
I made a little test version of my product and cart page. Using AJAX I passed the “id” parameter of the book to the method (AddToCart) and then reloaded the partial view. It’s just a few lines of jQuery and callback functions. Using “DateTime.Now” is a great way to test which HTML is reloading.

In the view (…Books/Test), the list of books are rendered in a table. The cart table is rendered aswell but as a partial page. Each row for the “book” table is given a button (add-to-cart). These buttons all have the same class (“btn”). When one of the buttons are clicked, it triggers this JavaScript code. “this.id” gets the id value, which I display in the table. If I could set this to hidden and still get the “id” value, that would be helpful. The first AJAX call sends this id to an action method in the controller which uses the id to find the book in the database, create a new cart object and set the parameters. Once this is done, the partial view, or “cart view” gets reloaded. Perfect! Sequential behaviour inside an asynchronous function.

IMPORTANT TERMS

Latency (Engineering) – Latency from a general point of view is a time delay between the cause and the effect of some physical change in the system being observed.

Separation of Concerns – is a design principle for separating a computer program into distinct sections such that each section addresses a separate concern. A concern is a set of information that affects the code of a computer program.

Test-Driven Development – is a software development process relying on software requirements being converted to test cases before software is fully developed, and tracking all software development by repeatedly testing the software against all test cases.

Transmission Control Protocol (TCP) – provides reliable, ordered, and error-checked delivery of a stream of octets (bytes) between applications running on hosts communicating via an IP network. Major internet applications such as the World Wide Web, email, remote administration, and file transfer rely on TCP, which is part of the Transport Layer of the TCP/IP suite. SSL/TLS often runs on top of TCP.

NPM – is a package manager for the JavaScript programming language maintained by npm, Inc. npm is the default package manager for the JavaScript runtime environment Node.js. It consists of a command line client, also called npm, and an online database of public and paid-for private packages, called the npm registry.
