Loading Images Faster 1 (Problem)

(01/09/2019): When a button is clicked, the old image fades out, the new image is set and faded in at the same time.

This creates a problem because an image doesn't download and render instantanously. During the "fade in" the image often renders.

Solution: HTML, CSS and jQuery

After learing more about this problem I came up with a solution.

I created a new div with an image element in the HTML and used the CSS stylesheet to hide the image. Then in the jquery I set this image early so that the image has time to cache and download.


Loading Images Faster 2

HTML add new image

At the bottom of the HTML code I added a div element.

Inside the div is an image with the id, "img2".


Loading Images Faster 3

CSS hide #img2

In the CSS style sheet I set the display attribute for the image to none.

xxxxxx


Loading Images Faster 4

Jquery set #img2

As soon as the "image URL" from the JSON data loads the hidden #img2 is set.

This executes before the old image fades out (700), giving the image time (700) to be downloaded and cached.

This fixed the issue that I was having and I used the technique because the images are relatively small and I only need to do one at a time.
