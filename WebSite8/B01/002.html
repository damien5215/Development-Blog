05/10/2020: Lost in Detail
Explaining a CRUD app. It’s almost finished and doing it has reminded me that it is a lot faster to do something complex than to explain something complex. This is probably part of the reason why Richard Feynman was so revered. Like all effective teachers, he enjoyed the challenge of explaining something. Really this is about being able to work with things in detail and broadly. Think globally, act locally. Whenever I get lost in detail, I think about Feynman and try to zoom out –

Clicking on the add button triggers the action method “Add()”. An instance of the view model for “adding books” is instantiated. The context object gets passed to the initialise method in the “add book” view model. The context object gets passed to the initialise method in the base “add book” view model. In the base “add book” view model, the select list for the “Authors” are retrieved from the database. In the “add book” view model, the two select lists for “Genres” and “Fiction/Non-Fiction” are retrieved from the database. Finally the “add book” view model is sent to the view and the HTML is rendered.

Once data has been entered by the USER and the “Save” button has been clicked, the “Book Add” view model data is posted to the “add post” method. This data is loaded to a variable named “book” and then the “AddGenre” method is called passing in the required parameters. The book is added to the context object, the changes are saved and the “detail” view for the new record is rendered.

Index() – Displays all books in database.
Detail() – Displays all info for a selected book.
Add() – Displays view for USER to enter data.
Add() POST – Saves data to database.
Edit() – Displays view for USER to edit record.
Edit() POST – Saves edited data to database.
Delete() – Displays view for a selected book.
Delete() POST – Deletes record from database.



03/10/2020: Rewriting C23
September flew by. Worked on explaining a CRUD app. Discovered and watched a dark comedy called Nighty Night (2004). Had some philosophical ideas and wrote about them. Watched Summer turn into Autumn. Learned how to play all of “Cry Me a River” (Julie London). The R rate went over 1. Rewrote a chapter in NC.

NC, C23 was a two page scene. It took place in an arcade and the dialogue was okay. I felt the scene was too short because the average length of a chapter/scene in NC is 5 pages. Whilst daydreaming I imagined the characters travelling (walk/train) from the arcade to my one of my favourite places to eat (Thai Express in Aldgate). This allowed me to extend the scene, add descriptions and more dialogue. I ended up with more structure (a beginning, middle and an end) and the scene is now 10 pages. NC is over 200 pages and I’m happier with it but I need to read it all again.
