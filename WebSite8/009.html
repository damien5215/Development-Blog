Bubble Sort: Pass array to sort2 method

An array of integers are declared: 4, 2, 3, 1

Then an instance of the bubble class is created.

The method "sort2" is called on this instance and the array is passed in.

The bubble class contains two main methods sort1 and sort2.

They are both implementations of the bubble sort algorithm.

The method sort2 prints to the console during every step of the calculation.

Migrations/Initial.cs

The intial class contains two methods, up and down.

The "Up" method contains code to update the database to be in sync wth the model that was used to generate this migration.

The "Down" method contains code to remove the updates that were applied in the "Up" method, so that the database will be in sync with the previous version of the model.
