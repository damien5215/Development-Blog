jQuery: Button click

Change images and text

When a button is clicked (target by class) this function is called.

The image fades out, all of the buttons (class) are set to black.

A variable (test1) gets the id of the button that was clicked (if button 2 is clicked, id = "butt1").

The id is then used to target the specific button and set it's colour to blue.

Next, a variable "a" gets the index of the button's id (id = "butt1" = index of 2).

This index is used in the other arrays to get the right text and image.

Finally the image is faded in (NOTE: the variable x1 set to 700).

Using jQuery tidied up the code (1 function rather than 11).

When I clicked on the error, line 28 of the HTML file was highlighted.

Then I realized that the button2() function that is called is function tht I had put in the Javascript file.

Console: Image Name Error

White-Space

This error was returning when I clicked on two of the buttons.

I soon realized that the names of the images had a white-space in them ("diagonal difference2.png").

By getting rid of the white-space ("diagonalDifference2.png") it fixed the issue.
